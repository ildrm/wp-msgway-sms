Send SMS notifications, OTP codes, and manage message logs seamlessly in WordPress with MsgWay SMS.

== Description ==

MsgWay SMS is a powerful WordPress plugin that enables you to send SMS notifications, one-time passwords (OTP), and promotional messages to your users. Integrated with WordPress, WooCommerce, and various messaging services, it offers a flexible solution for user engagement and communication.

### Key Features
- **SMS Notifications**: Send welcome messages, password reset codes, post publication alerts, and more.
- **OTP Support**: Secure user authentication with customizable one-time password codes.
- **WooCommerce Integration**: Notify customers about order confirmations, status changes, abandoned carts, and promotional offers.
- **Message Logs**: Track and filter sent messages with a detailed logs table, including status and response details.
- **Multiple Providers**: Choose from various SMS and messaging services like SMS 3000x, WhatsApp, and more.
- **Multilingual**: Supports Persian (Farsi) and English for messages and admin interface.
- **Customizable Settings**: Configure API keys, message templates, and provider settings easily.

Whether you're running a blog, an e-commerce store, or a membership site, MsgWay SMS helps you stay connected with your audience through timely and relevant notifications.

== Installation ==

1. **Install the Plugin**:
   - Upload the `wp-msgway-sms` folder to the `/wp-content/plugins/` directory, or
   - Install the plugin directly through the WordPress admin panel under Plugins > Add New.
2. **Activate the Plugin**:
   - Go to Plugins in your WordPress admin panel and activate "MsgWay SMS".
3. **Configure Settings**:
   - Navigate to the "MsgWay SMS" menu in the WordPress admin panel.
   - Enter your API key from ildrm.com and configure general settings, OTP, WordPress, and WooCommerce options as needed.
4. **Test the Setup**:
   - Send a test message or OTP to verify the configuration.
   - Check the "Logs" tab to ensure messages are being logged correctly.

== Frequently Asked Questions ==

= How do I get an API key? =
Visit [ildrm.com](https://ildrm.com) to sign up and obtain your API key. Enter it in the plugin's settings under the "General" tab.

= Does this plugin support WooCommerce? =
Yes, MsgWay SMS integrates seamlessly with WooCommerce. You can send notifications for order confirmations, status changes, abandoned carts, and promotional campaigns.

= Can I customize the OTP length and expiration time? =
Absolutely! In the "OTP" tab, you can set the number of digits for OTP codes and their expiration time (in seconds).

= What messaging services are supported? =
The plugin supports multiple providers, including SMS 3000x, SMS 2000x, SMS 9000x, WhatsApp, and Gap. You can select your preferred provider in the settings.

= How can I view sent messages? =
Go to the "Logs" tab in the plugin settings to view a detailed table of sent messages, including type, mobile number, status, and response details. You can filter logs by message type, status, or date.

= Is Persian (Farsi) supported? =
Yes, the plugin supports Persian for both the admin interface and message content. You can select the language in the general settings.
